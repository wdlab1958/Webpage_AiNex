Architecture Guide

AI Consulting Assistant Platform: 기술 아키텍처 및 워크플로우 설계서

Version 1.2 Last Updated: 2025.12.02

1. 플랫폼 개요

AiNex는 기업의 AI 전환(AX)을 지원하는 멀티 에이전트 기반 컨설팅 플랫폼입니다. 인간 컨설턴트와 멀티 AI 에이전트가 역할 기반 협업(Role-based Collaboration)을 수행하여 전략 수립부터 인프라 구축까지의 과정을 자동화합니다.

핵심 특징
  • 보안 중심 설계 (Local LLM)
  • 5개 전문 에이전트 협업
  • 가중치 기반 정량 평가 엔진
  • 5단계 컨설팅 프레임워크 자동화

기술 스택

  • FrontendHTML5, CSS3, JavaScript
  • BackendFastAPI (Python 3.12)
  • LLM EngineOllama (llama3.1:8b)
  • Agent FrameworkLangGraph, CrewAI, AutoGen
  • DatabaseSQLite + aiosqlite
  • Embeddingnomic-embed-text

2. 하이브리드 LLM 아키텍처

AiNex은 보안을 최우선으로 하는 하이브리드 접근법을 채택합니다. 민감한 기업 데이터는 사내 폐쇄망의 Local LLM(Ollama)에서 처리하며, 일반적인 추론 및 성능 극대화가 필요한 경우 온라인 LLM과 연동할 수 있는 지능형 라우팅을 지원합니다.

3. 멀티 에이전트 프레임워크

LangGraph

에이전트 워크플로우 상태 관리 및 노드 간 오케스트레이션을 담당하여 복잡한 순환 그래프 구조를 관리합니다.

CrewAI

역할 기반의 에이전트 협업 체계를 구축하며, 각 에이전트 간의 작업을 분배하고 결과를 취합합니다.

AutoGen

인간 컨설턴트와의 자율적인 대화 루프를 생성하고 피드백을 시스템에 즉각 반영하는 인터페이스를 구현합니다.

5. 가중치 기반 분석 시스템

컨설턴트가 입력한 기업 데이터를 바탕으로 실시간 가중치 알고리즘을 실행합니다. IT 인프라 인프라, 데이터 자산, 구성원 역량 점수를 합산하여 최종 성숙도와 ROI를 도출합니다.

# 성숙도 점수 계산 알고리즘 예시 def calculate_maturity(company): score = 2.0 # 기본값 if company.executive_support >= 4: score += 1.0 if company.gpu_available: score += 0.5 if company.data_scientists > 0: score += 1.0 return score

개인정보 처리방침

AiNex는 사용자의 개인정보를 소중하게 생각하며, 관련 법규를 준수합니다. 본 방침은 당사 플랫폼 서비스를 이용함에 있어 제공되는 개인정보가 어떻게 처리되는지 안내합니다.

1. 수집하는 개인정보 항목

컨설팅 서비스 제공을 위해 다음과 같은 정보를 수집할 수 있습니다: 기업명, 담당자 이름, 연락처, 이메일 주소, 기업 인프라 현황 데이터 등.

2. 개인정보 수집 및 이용 목적

수집된 정보는 AI 전환 컨설팅 리포트 생성, 맞춤형 아키텍처 설계, 고객 지원 및 서비스 개선 목적으로만 사용됩니다.

3. 보안 및 데이터 보호 (Local LLM)

AiNex는 보안형 하이브리드 LLM 아키텍처를 기반으로 합니다. 핵심 기업 데이터는 사내 폐쇄망 내 Local LLM(Ollama)에서 처리되어 외부 유출을 원천적으로 차단합니다.

4. 개인정보의 보유 및 이용기간

원칙적으로 개인정보 수집 및 이용 목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다. 단, 관계 법령에 의해 보존할 필요가 있는 경우 일정 기간 보관합니다.

서비스 이용약관

본 약관은 AiNex 플랫폼이 제공하는 모든 서비스 및 관련 기능의 이용 조건과 절차에 관한 사항을 규정합니다.

1. 서비스의 목적

당사는 최신 AI 기술을 활용하여 기업의 성공적인 AI 전환(AX)을 돕기 위한 분석 및 설계 가이드를 제공합니다.

2. AI 생성 결과물의 책임 제한

AiNex의 멀티 에이전트 시스템이 생성하는 컨설팅 리포트 및 아키텍처 가이드는 데이터에 기반한 제안이며, 최종적인 기술 도입 및 비즈니스 결정은 사용자의 전문가 검토를 거쳐야 합니다. 인공지능 모델의 특성상 생성된 정보의 완전한 정확성을 보장하지 않을 수 있습니다.

3. 지식재산권

플랫폼 내의 모든 디자인, 로직, 에이전트 프레임워크에 대한 권리는 AiNex에 귀속됩니다. 사용자는 제공받은 리포트를 해당 기업의 내부 목적으로만 사용할 수 있습니다.
