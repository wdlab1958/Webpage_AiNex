Core Security Technologies

Local LLM Enforcement

Ollama 기반의 경량화된 모델(Llama-3, Mistral)을 온프레미스 서버에 구축합니다. 인터넷 연결이 차단된 상태에서도 추론이 가능하며, 프롬프트나 결과 데이터가 외부로 절대 노출되지 않습니다.

$ ollama run ainex-secure-model

Intelligent Traffic Router

사용자의 요청을 분석하여 보안 등급을 실시간으로 판별합니다. 민감 정보가 포함된 요청은 즉시 로컬 에이전트로 우회시키고, 일반적인 정보 요약이나 번역은 클라우드 API를 활용하여 효율성을 극대화합니다.

End-to-End Encryption

모든 데이터 전송 구간에 AES-256 암호화를 적용하며, AI 분석 시에도 개인 식별 정보(PII)를 자동으로 마스킹하는 전처리 계층을 운영합니다.

Compliance & Trust

AiNex는 금융권 및 민간 대기업의 엄격한 보안 요건을 충족합니다. 망분리 가이드라인 준수 및 클라우드 보안인증(CSAP) 대응이 가능합니다.

  • 개인정보보호법(PIPA) 컴플라이언스
  • 금융감독원 망분리 예외 조항 대응
  • ISO/IEC 27001 정보보호 관리체계
